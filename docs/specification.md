# ポケモンチャンピオンズ対戦支援ツール 仕様書

## 概要

ポケモンチャンピオンズの対戦前に、相手パーティをもとにAIが選出予想を生成し、各ポケモンの型情報（技・持ち物・特性・性格・個体値・能力ポイント）を表示する個人利用ツール。
